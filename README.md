# ZERO HOUR — Retro Arcade Shooter

75초 안에 크리스털을 모으고, 콤보를 쌓고, 보스 **Core Warden**을 격파하는 레트로 드론 아케이드 슈터입니다.
순수 HTML/CSS/JavaScript + Canvas 2D로 만들어져 빌드 도구 없이 브라우저에서 바로 실행됩니다.

> 내부 코드명: **Scoreyard**

## 실행 방법

별도 설치나 빌드가 필요 없습니다.

| 파일 | 설명 |
|---|---|
| `Scoreyard.html` | 메인 진입점 — `js/` 폴더의 스크립트를 로드 (개발용) |
| `Scoreyard Standalone.html` | 모든 코드가 번들된 단일 파일 버전 — 파일 하나로 어디서나 실행 |

```
# 그냥 브라우저로 열거나
start Scoreyard.html

# 로컬 서버로 실행 (선택)
npx serve .
```

## 게임 방법

- **목표**: 제한 시간 75초 동안 최대한 높은 점수를 획득
- **조작**: `WASD` / 방향키 — 이동 · **발사는 자동** (가장 가까운 적 조준)
- **모바일**: 화면 드래그로 가상 조이스틱 조작
- **체력(HULL)**: ◆ 3칸 — 기뢰·보스 탄막·보스 접촉에 피격 시 1칸 감소, 0이 되면 게임 오버
- **보스**: 남은 시간 20초에 **CORE WARDEN** 등장 — 격파 시 대량 보너스
- 우측 상단 🔊 버튼으로 음소거 전환

### 게임 모드

| 모드 | 설명 |
|---|---|
| **DAILY CHALLENGE** | UTC 날짜 기반 시드 — 오늘은 전 세계 모두 같은 맵에서 점수 경쟁. 베스트 기록과의 실시간 페이스 비교(`+N vs best`)가 HUD에 표시됩니다 |
| **FREE PLAY** | 매번 무작위 시드 — 부담 없는 연습 모드 |

## 점수 시스템

| 행동 | 점수 |
|---|---|
| 크리스털 수집 | `10 + 콤보` (연속 수집 시 콤보 증가, 2.6초 내 미수집 또는 피격 시 리셋) |
| 기뢰(Mine) 파괴 | 25 |
| 바위(Rock) 파괴 | 40 + 크리스털 4~5개 드랍 (45% 확률로 파워업 드랍) |
| 보스 명중 | 발당 5 |
| **보스 격파** | **1,500** + 크리스털 14개 폭발 드랍 |

×2 SCORE 파워업 적용 중에는 모든 점수가 2배가 됩니다.

## 파워업

| 글리프 | 이름 | 효과 | 지속 |
|:---:|---|---|:---:|
| `M` | MAGNET | 주변 크리스털을 끌어당김 | 7초 |
| `S` | SHIELD | 피격 1회 무효화 | 소모형 |
| `T` | SLOW-MO | 적·탄막 속도 감소 | 5초 |
| `×2` | ×2 SCORE | 획득 점수 2배 | 7초 |
| `»` | BOOST | 이동 속도 1.5배 | 6초 |
| `Ψ` | SPREAD | 3방향 발사 | 7초 |
| `+5` | +5 SEC | 남은 시간 +5초 | 즉시 |

    ├── main.js               # UI 글루 — 게임 루프, HUD, 화면 전환, 기록, 결과 공유, 터치 조이스틱

## 기술 노트

- **시드 RNG**: 데일리 챌린지는 `daily-YYYY-MM-DD`(UTC) 문자열을 xmur3 해시 → mulberry32로 변환해 사용합니다. 같은 날짜에는 누구나 동일한 크리스털·적·파워업 배치를 받아 공정한 점수 경쟁이 가능합니다.
- **기록 저장**: 올타임 베스트, 데일리 베스트, 초당 점수 페이스가 IndexedDB(`scoreyard` DB)에 로컬 저장됩니다. 서버 통신은 없습니다.
- **결과 공유**: 게임 오버 화면의 `COPY RESULT`로 결과 텍스트를 클립보드에 복사할 수 있으며, 개인정보는 포함되지 않습니다.
- **트윅 패널**: React 18 + Babel Standalone(CDN)으로 구동되는 개발용 패널로, 게임 시간(60~120초), 적 스폰 속도, 파티클 강도, 화면 흔들림을 실시간 조정할 수 있습니다. 게임 본체는 React에 의존하지 않습니다.
- **연출**: CRT 스캔라인 + 비네트 오버레이, 히트스톱(freeze), 화면 흔들림, 파티클 버스트로 레트로 아케이드 타격감을 구현했습니다.
